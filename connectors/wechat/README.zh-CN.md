# WeChat Connector

[English](README.md)

WeChat Connector 用于发布将 xAgent 接入微信 IM 场景的连接器二进制包。

## Release Tag

请使用连接器独立的 Release tag：

```text
wechat-v0.0.1
```

不要把 WeChat Connector 附件发布到通用的 `v0.0.1` Release 中，因为本仓库后续可能会发布多个不同功能的连接器。

## 发布附件

常见附件示例：

```text
xagent-wechat-connector-v0.0.1-linux-amd64.tar.gz
xagent-wechat-connector-v0.0.1-linux-arm64.tar.gz
xagent-wechat-connector-v0.0.1-darwin-amd64.tar.gz
xagent-wechat-connector-v0.0.1-darwin-arm64.tar.gz
SHA256SUMS
```
